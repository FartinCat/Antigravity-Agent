# Workflow: TDD Cycle

**Objective**: Ensure high code quality and functional correctness through testing.

## Steps
1. **Red**: Write a failing test for the desired functionality.
2. **Green**: Implement the minimal code necessary to make the test pass.
3. **Refactor**: Clean up the code while ensuring the test stays green.
4. **Verify**: Run the entire test suite to ensure no regressions.

## Tools Involved
- Test runner (project-specific)
- `tdd-guide` agent
- `refactor` skill
