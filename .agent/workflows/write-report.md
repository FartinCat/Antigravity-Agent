# Workflow: Write Report

**Objective**: Execute a pipeline for generating high-quality academic or technical reports.

## Execution Sequence

1. **Outline**: Invoke the `planner` to create a chapter-by-chapter outline.
2. **Draft**: Apply the `scientific-writing` skill to ensure objective tone and banned patterns are respected.
3. **Format**: Invoke the `latex-bib-manager` to clean citations, sort bibliographies, and enforce strict float anchors.
4. **Finalize**: Generate the final PDF or document output.
