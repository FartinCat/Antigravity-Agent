# Skill: Deep Scan

**Objective**: Comprehensive situational awareness of the repository.

## Execution Protocol
1. **Tree Exploration**: List all files and directories recursively to understand the project architecture.
2. **Resource Identification**: Scan for non-code assets (images, data files, configurations, documentation).
3. **Dependency Mapping**: Read package files (e.g., `package.json`, `requirements.txt`, `.bib`) to understand the technical stack.
4. **Context Linking**: Identify relationships between different modules (e.g., how the UI interacts with the API).
5. **Memory Update**: Update the internal project "mental map" to ensure all future suggestions are context-aware.

## Standard Trigger
Should be executed at the beginning of every session or when a major structural change is detected.
